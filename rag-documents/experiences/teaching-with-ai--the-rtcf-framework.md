## The RTCF Framework

The difference between a mediocre AI response and a brilliant one often comes down to how you ask. RTCF is a simple framework that transforms vague requests into precise prompts.

R - Role

Who should AI be?

"You are an experienced professional who specialises in..."

T - Task

What should it do?

"Create a summary of... / Draft an email about... / Analyse..."

C - Context

What background does it need?

"The audience is... The deadline is... This is for..."

F - Format

How should output look?

"4-point scale with descriptors, table format..."

Common Mistakes to Avoid

**Too vague:** "Help me with my lecture"

Fix: Be specific about what help you need

**No context:** "Write quiz questions"

Fix: Include topic, level, purpose

**No format:** "Explain this concept"

Fix: Specify length, structure, style

**One-shot thinking:** Taking first output as final

Fix: Iterate. First output is a draft
